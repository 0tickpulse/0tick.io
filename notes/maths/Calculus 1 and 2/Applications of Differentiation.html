Applications of Differentiation

Differentiation is a powerful tool in mathematics, and it has many applications in real life.

Mean Value Theorem​

Imagine a function f(x)f(x)f(x) that is continuous over the closed interval [a,b][a, b][a,b], but is differentiable over the open interval (a,b)(a, b)(a,b). So, at exactly aaa and bbb, no derivative exists but there is still a point there.

What the mean value theorem tells us is that at some point between aaa and bbb, the slope at that point is equal to the average slope over the whole interval.

Formally it is written like this:

∃c∈(a,b) such that dfdx(c)=f(b)−f(a)b−a\exists c \in (a, b) \, \text{such that} \, \frac{df}{dx}(c) = \frac{f(b) - f(a)}{b - a}∃c∈(a,b)such thatdxdf​(c)=b−af(b)−f(a)​

Where ccc is some point between aaa and bbb.

Note: when the mean-value theorem does not apply, it doesn't mean that it's necessarily false. It just means that it's uncertain.

Proof​

Taking the expression above (which is the average slope of the function), we get the slope of a line that intersects (a,f(a))(a, f(a))(a,f(a)) and (b,f(b))(b, f(b))(b,f(b)).

Let's define a new function g(x)=f(x)−mxg(x) = f(x) - mxg(x)=f(x)−mx, where mmm is a constant. Given that fff is continuous and differentiable, ggg is also continuous and differentiable.

g(a)=g(b)  ⟺  f(a)−ma=f(b)−mbg(a) = g(b) \iff f(a) - ma = f(b) - mbg(a)=g(b)⟺f(a)−ma=f(b)−mb

Solving for mmm gives:

f(a)−ma=f(b)−mbm=f(b)−f(a)b−a\begin{align*} f(a) - ma &= f(b) - mb \\ m &= \frac{f(b) - f(a)}{b - a} \end{align*}f(a)−mam​=f(b)−mb=b−af(b)−f(a)​​

According to Rolle's theorem, there must be a point on the graph where it is flat (horizontal), that is, (∃c∈(a,b)) dgdx(c)=0(\exists c \in (a, b)) \, \frac{dg}{dx}(c) = 0(∃c∈(a,b))dxdg​(c)=0.

Based on the definition of g(x)g(x)g(x):

dgdx(c)=dfdx(c)−m=0dfdx(c)=mdfdx(c)=f(b)−f(a)b−a\begin{align*} \frac{dg}{dx}(c) &= \frac{df}{dx}(c) - m = 0 \\ \frac{df}{dx}(c) &= m \\ \frac{df}{dx}(c) &= \frac{f(b) - f(a)}{b - a} \end{align*}dxdg​(c)dxdf​(c)dxdf​(c)​=dxdf​(c)−m=0=m=b−af(b)−f(a)​​

Example Application: Speed Limits​

Consider a car that travels from point AAA to point BBB in an hour, where the distance is 75 km75 \text{ km}75 km.

Imagine the speed limit is 60 km/h60 \text{ km/h}60 km/h. You would think that the car's velocity must be measured throughout the journey to ensure that it doesn't exceed the speed limit. Instead, you can just use the average velocity to determine if the car has exceeded the speed limit.

To do this, you can use the mean value theorem. The average velocity is:

vavg=r(b)−r(a)b−a=751=75 km/hv_{\text{avg}} = \frac{\vec{r}(b) - \vec{r}(a)}{b - a} = \frac{75}{1} = 75 \text{ km/h}vavg​=b−ar(b)−r(a)​=175​=75 km/h

We don't need to know the exact position/velocity/etc of the car at all times, because using the MVT, we can determine that the car must have exceeded the speed limit at some point.

Thus, the car must have exceeded the speed limit at some point between AAA and BBB, and the driver can receive a ticket.

Extreme Value Theorem​

Just like the mean value theorem, the extreme value theorem is another important theorem helping us understand the behavior of functions. While these theorems can sound like common sense, it's important to be able to rigorously prove them such that we are absolutely certain they are true. Imagine receiving a speed ticket because of a theorem that isn't actually true!

The extreme value theorem states that if a function f(x)f(x)f(x) is continuous over a closed interval [a,b][a, b][a,b], then there must be a maximum and minimum value of f(x)f(x)f(x) over that interval.

This means that if you have a continuous function over a closed interval, you can be certain that there is a point where the function reaches its maximum and minimum values.

Formally, the theorem states:

∃c,d∈[a,b] such that f(c)≥f(x)≥f(d) ∀x∈[a,b]\exists c, d \in [a, b] \, \text{such that} \, f(c) \geq f(x) \geq f(d) \, \forall x \in [a, b]∃c,d∈[a,b]such thatf(c)≥f(x)≥f(d)∀x∈[a,b]

For a function that isn't continuous, the theorem doesn't apply.

Concavity and Inflection Points​

Concavity is a property of a function that describes how the function is curving.

If the function is curving upwards, it is concave up. If the function is curving downwards, it is concave down.

The easiest example of this is a parabola. A parabola like y=x2y = x^2y=x2 is concave up at x=0x = 0x=0, while a parabola like y=−x2y = -x^2y=−x2 is concave down at x=0x = 0x=0.

To find the concavity of a function, we can use the second derivative;

  • If d2fdx2>0\frac{d^2f}{dx^2} > 0dx2d2f​>0, the function is concave up.
  • If d2fdx2<0\frac{d^2f}{dx^2} < 0dx2d2f​<0, the function is concave down.

Identifying Increasing and Decreasing Intervals​

The derivative of a function tells us the slope of the function at any point.

If the derivative is positive, the function is increasing. If the derivative is negative, the function is decreasing.

As such, if the derivative is positive over an interval, the function is increasing over that interval, and vice versa.

For example, consider the function f(x)=x5−3x4f(x) = x^5 - 3x^4f(x)=x5−3x4:

Using Critical Points​

Critical points can also be used to determine increasing and decreasing intervals.

Recall that critical points are points where the derivative is 000 or undefined.

If a point is a maximum, the function is decreasing before and after the point, and vice versa for a minimum. We can use this to determine increasing and decreasing intervals.

Let's consider the same function f(x)=x5−3x4f(x) = x^5 - 3x^4f(x)=x5−3x4.

Solving the derivative 5x4−12x3=05x^4 - 12x^3 = 05x4−12x3=0 gives us x=0x = 0x=0 and x=125x = \frac{12}{5}x=512​. These are the critical points of the function.

To find the minimum and maximum points, we can use the second derivative.

The second derivative of f(x)f(x)f(x) is:

d2fdx2(x)=ddx(5x4−12x3)=20x3−36x2\frac{d^2f}{dx^2}(x) = \frac{d}{dx}(5x^4 - 12x^3) = 20x^3 - 36x^2dx2d2f​(x)=dxd​(5x4−12x3)=20x3−36x2

At x=0x = 0x=0, the second derivative is 000, so the test is inconclusive.

At x=125x = \frac{12}{5}x=512​, the second derivative is 20(125)3−36(125)2=69.1220 \left(\frac{12}{5}\right)^3 - 36 \left(\frac{12}{5}\right)^2 = 69.1220(512​)3−36(512​)2=69.12. Since the second derivative is positive, the function is concave up, and x=125x = \frac{12}{5}x=512​ is a minimum point.

This means that the function is decreasing before x=125x = \frac{12}{5}x=512​ and increasing after x=125x = \frac{12}{5}x=512​.

At inconclusive points, the curve "flips" from increasing to decreasing or vice versa. So if the function is increasing at [0,125][0, \frac{12}{5}][0,512​], it must be decreasing at [−∞,0][-\infty, 0][−∞,0].

Finding Global Extrema using Candidate Tests​

Global extrema are the maximum and minimum points of a function over a closed interval. It's not very different from finding local extrema, but it requires a bit more work.

We can take advantage of the extreme value theorem. If a function is continuous over a closed interval, it must have a maximum and minimum point.

For example, consider the function f(x)=8ln⁡(x)−x2f(x) = 8\ln(x) - x^2f(x)=8ln(x)−x2:

Recall that ln⁡(x)\ln(x)ln(x) is only defined for x>0x > 0x>0, so the domain of the function is x>0x > 0x>0. For the purposes of this example, we'll consider the interval x∈[1,4]x \in [1, 4]x∈[1,4].

To find the global extrema of the function, we need to find the critical points of the function.

The derivative of f(x)f(x)f(x) is:

dfdx(x)=ddx(8ln⁡(x)−x2)=8x−2x\frac{df}{dx}(x) = \frac{d}{dx}(8\ln(x) - x^2) = \frac{8}{x} - 2xdxdf​(x)=dxd​(8ln(x)−x2)=x8​−2x

8x−2x=0  ⟹  8−2x2=0  ⟹  x=±2\frac{8}{x} - 2x = 0 \implies 8 - 2x^2 = 0 \implies x = \pm 2x8​−2x=0⟹8−2x2=0⟹x=±2

Thus, the critical points of the function are x=2x = 2x=2 and x=−2x = -2x=−2.

To find the global maximum and minimum points, we can make use of the extreme value theorem.

The function is continuous over the interval [1,4][1, 4][1,4], so it must have a maximum and minimum point. The maximum point is at x=2x = 2x=2, and the minimum point is at the endpoints of the interval.

What we've done here is known as the candidate test. We find the critical points of the function, and then test these points to determine if they are maximum or minimum points.

Optimization​

Optimization refers to finding the maximum or minimum value of a function. This is a common problem in mathematics, and it has many real-world applications.

For example, consider a company that wants to maximize its profit. The company can use optimization to determine the best price to sell its products.

To solve an optimization problem, we need to follow these steps:

  1. Define the function to be optimized.
  2. Identify the constraints of the problem.
  3. Find the critical points of the function.
  4. Use the candidate test to determine the maximum or minimum points.

In other words, it combines everything we've learned so far.

Example Problem - Maximizing Profit​

A company wants to maximize its profit. The company's revenue is given by R(x)=100x−x2R(x) = 100x - x^2R(x)=100x−x2, and its cost is given by C(x)=50+10xC(x) = 50 + 10xC(x)=50+10x, where xxx is the number of units sold. Find the number of units the company should sell to maximize its profit.

To solve this problem, we need to find the profit function, which is the difference between the revenue and cost functions.

P(x)=R(x)−C(x)=(100x−x2)−(50+10x)=90x−x2−50=−x2+90x−50\begin{align*} P(x) &= R(x) - C(x) \\ &= (100x - x^2) - (50 + 10x) \\ &= 90x - x^2 - 50 \\ &= -x^2 + 90x - 50 \end{align*}P(x)​=R(x)−C(x)=(100x−x2)−(50+10x)=90x−x2−50=−x2+90x−50​
