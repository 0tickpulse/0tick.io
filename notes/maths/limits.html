Limits

Limits are a fundamental concept in calculus. They are used to define the derivative and the integral, and are used to define continuity.

In essence, a limit asks the question: "What value does a function approach as it gets closer and closer to a certain point?"

Notation​

The limit of a function f(x)f(x)f(x) as xxx approaches aaa is denoted by:

lim⁡x→af(x)\lim_{x \to a} f(x)x→alim​f(x)

This is read as "the limit of f(x)f(x)f(x) as xxx approaches aaa." It asks, "what value does f(x)f(x)f(x) approach as xxx gets closer and closer to aaa?"

Examples​

For instance, consider the function f(x)=∣1x∣f(x) = \left| \frac{1}{x} \right|f(x)=​x1​​. As xxx gets closer and closer to 0, f(x)f(x)f(x) gets larger and larger. This can also be viewed on a graph:

Notice how while it is impossible to actually reach 0, the function gets larger and larger as xxx gets closer and closer to 0. This is why the limit of f(x)f(x)f(x) as xxx approaches 0 is ∞\infty∞.

Written mathematically:

lim⁡x→0∣1x∣=∞\lim_{x \to 0} \left| \frac{1}{x} \right| = \inftyx→0lim​​x1​​=∞

One-Sided Limits​

Let's say we want to get the limit of a function. This time, we're not going to get the function itself, but rather just the graph.

If we try to approach x=−2x = -2x=−2, you see that there's a massive (infinite) gap between the two sides. And you approach different values depending on which side you're coming from.

This is why we have the concept of one-sided limits.

The limit of f(x)f(x)f(x) as xxx approaches aaa from the left is denoted by:

lim⁡x→a−f(x)\lim_{x \to a^-} f(x)x→a−lim​f(x)

And similarly, the limit of f(x)f(x)f(x) as xxx approaches aaa from the right is denoted by:

lim⁡x→a+f(x)\lim_{x \to a^+} f(x)x→a+lim​f(x)

In this case, using the graph alone, we can see that:

lim⁡x→−2−f(x)=−1lim⁡x→−2+f(x)=∞\begin{align*} \lim_{x \to -2^-} f(x) &= -1 \\ \lim_{x \to -2^+} f(x) &= \infty \end{align*}x→−2−lim​f(x)x→−2+lim​f(x)​=−1=∞​

Estimating Limits​

There are many ways to estimate a limit. In fact, we just approximated one by looking at the graph above.

Graph Visualization​

Graphing the function and seeing what value it approaches as xxx gets closer and closer to the point in question is a good way to estimate a limit.

Example 1​

lim⁡x→0(sin⁡(x)x)\lim_{x \to 0} \left(\frac{\sin(x)}{x}\right)x→0lim​(xsin(x)​)

sin⁡(0)0=00\frac{\sin(0)}{0} = \frac{0}{0}0sin(0)​=00​

Oh no! We can't divide by 0! This is a problem. However, we can still try to estimate the limit by graphing the function:

Notice that while there's a gap at x=0x = 0x=0, the function seems to approach 1 as xxx gets closer and closer to 0. This is why:

lim⁡x→0(sin⁡(x)x)=1\lim_{x \to 0} \left(\frac{\sin(x)}{x}\right) = 1x→0lim​(xsin(x)​)=1

This is not exactly why, but it's good enough for now. A better explanation like using the squeeze theorem will be given later.

Tables​

Another way to estimate a limit is by creating a table of values. This is especially useful when the function is not graphable, or you don't have access to a graphing calculator.

Example 2​

Let's say this table is given:

xxx3.93.93.93.993.993.993.9993.9993.9994.0014.0014.0014.014.014.014.14.14.1

We can estimate the limit of f(x)f(x)f(x) as xxx approaches 4 by looking at the values of f(x)f(x)f(x) as xxx gets closer and closer to 4.

Generally, look for these things when constructing or analyzing a table like this:

  • The values chosen for xxx should get closer and closer to the value in question.
  • The values of f(x)f(x)f(x) should be getting closer and closer to a single value.
  • If the values of f(x)f(x)f(x) are getting closer and closer to a single value, then that value is our estimate for the limit.
  • The limit can be two-sided, and that's shown by the values of f(x)f(x)f(x) getting closer and closer to a single value from both sides.

General Procedure​

For this limit:

lim⁡x→af(x)\lim_{x \to a} f(x)x→alim​f(x)

You can apply this general procedure:

  1. Try to substitute in x=ax = ax=a. If you get a number, then that's probably the limit.
  2. If you get b0\frac{b}{0}0b​ where b≠0b \neq 0b=0, then you probably hit an asymptote.
  3. If you get 00\frac{0}{0}00​, then you got an indeterminate form.
  4. If it's indeterminate, try to rewrite the limit by factoring, rationalizing, or using trigonometric identities.
  5. If all else fails, try to estimate the limit using a graph or a table.

The function f(x)f(x)f(x) is defined as follows:

Combining Functions​

If you have a limit of a function that's a combination of other functions, you can use the limit laws to simplify the limit.

Sum and Difference​

The limit of sum is the sum of limits.

lim⁡x→a(f(x)+g(x))=lim⁡x→af(x)+lim⁡x→ag(x)\lim_{x \to a} \left(f(x) + g(x)\right) = \lim_{x \to a} f(x) + \lim_{x \to a} g(x)x→alim​(f(x)+g(x))=x→alim​f(x)+x→alim​g(x)

Products​

The limit of a product is the product of limits.

lim⁡x→a(f(x)⋅g(x))=lim⁡x→af(x)⋅lim⁡x→ag(x)\lim_{x \to a} \left(f(x) \cdot g(x)\right) = \lim_{x \to a} f(x) \cdot \lim_{x \to a} g(x)x→alim​(f(x)⋅g(x))=x→alim​f(x)⋅x→alim​g(x)

Composition​

This is a bit more interesting. Let's say you want to find the limit of f(g(x))f(g(x))f(g(x)) as xxx approaches aaa. This is called a composition of functions, where you first apply g(x)g(x)g(x) and then apply f(x)f(x)f(x) to the result.

For this to work, two conditions must be met:

  1. A value LLL must exist such that lim⁡x→ag(x)=L\lim_{x \to a} g(x) = Llimx→a​g(x)=L.
  2. The function f(x)f(x)f(x) must be continuous at LLL.

If these conditions are met, then:

lim⁡x→af(g(x))=f(lim⁡x→ag(x))=f(L)\lim_{x \to a} f(g(x)) = f\left(\lim_{x \to a} g(x)\right) = f(L)x→alim​f(g(x))=f(x→alim​g(x))=f(L)

Squeeze Theorem​

The squeeze theorem is a powerful tool for finding limits. It's especially useful when you have a function that's hard to evaluate directly.

Let's use an analogy: Alice, Bob, and Charlie all have different heights. Bob is taller than Alice, and Charlie is taller than Bob.

Mathematically: Alice’s height≤Bob’s height≤Charlie’s height\text{Alice's height} \leq \text{Bob's height} \leq \text{Charlie's height}Alice’s height≤Bob’s height≤Charlie’s height.

Let's say we know Alice's height to be 160cm160\text{cm}160cm and Charlie's height to also be 160cm160\text{cm}160cm. Since Bob's height must be between Alice's and Charlie's, we can conclude that Bob's height is also 160cm160\text{cm}160cm.

This is the essence of the squeeze theorem. If you have a function f(x)f(x)f(x) that's always between two other functions g(x)g(x)g(x) and h(x)h(x)h(x), and the limits of g(x)g(x)g(x) and h(x)h(x)h(x) as xxx approaches aaa are the same, then the limit of f(x)f(x)f(x) as xxx approaches aaa is also the same.

Remember our function sin⁡(x)x\frac{\sin(x)}{x}xsin(x)​? We estimated the limit to be 1 by graphing the function. However, we can also use the squeeze theorem to rigorously prove that the limit is 1.

As we know, trigonomic functions can be modeled by the unit circle. We will use this to visualize this proof.

Formal things​

Our definition of limits is simple and intuitive, but it's not rigorous. We can't just say "the function gets closer and closer to a value" - everything in Mathematics needs to be defined precisely.

One way to define limits rigorously is using the (ε(\varepsilon(ε-δ)\delta)δ) definition of limits, or the "epsilon-delta" definition of limits.

Graphically, this is what it looks like:

Suppose we want to find out what this function approaches as xxx approaches 000. We've already done this by looking at values around x=0x = 0x=0. In this case, we have a range of xxx values around 000 shown by the red lines, and a range of f(x)f(x)f(x) values around f(0)f(0)f(0) shown by the blue lines. The (ε(\varepsilon(ε-δ)\delta)δ) definition of limits tells us that if we decrease the size of the range of xxx values, the size of the range of f(x)f(x)f(x) values also decreases.

The key point being that the size of the range of f(x)f(x)f(x) values can decrease however much we want (try moving that slider).

How do ε\varepsilonε and δ\deltaδ fit into this? Take this example:

lim⁡x→af(x)=L\lim_{x \to a} f(x) = Lx→alim​f(x)=L

Think of any distance from LLL, and call it ε\varepsilonε.

What it means in terms of the limit is that you will always be able to find a range of inputs around aaa, some distance δ\deltaδ away from aaa, such that the outputs of f(x)f(x)f(x) will always be within ε\varepsilonε of LLL.

And this is true for any ε\varepsilonε you choose, no matter how small.

Here's a formal way to write this to impress your friends:

(∀ε>0)(∃δ>0)(∀x∈R)(0<∣x−a∣<δ  ⟹  ∣f(x)−L∣<ε)(\forall \varepsilon > 0)(\exists \delta > 0)(\forall x \in \mathbb{R})(0 < |x - a| < \delta \implies |f(x) - L| < \varepsilon)(∀ε>0)(∃δ>0)(∀x∈R)(0<∣x−a∣<δ⟹∣f(x)−L∣<ε)
